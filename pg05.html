Descrição - Vídeo 02

  • CRUD e HTTP

  • Cada uma dessas operações do CRUD está associada a um método HTTP especifíco
  • POST: Usamos para criar dados, como uma nova publicação no blog
  • GET: É para buscar ou listar informações, como todas as postagens de um blog
  • PUT ou PATCH: Serve para atuaizar algo, por exemplo, o titulo de uma publicação
  • DELETE: Como o nome sugere, é para excluir algum dado
